12 Comandos mais comuns do Git

git init

🚀

Inicia um novo projeto com o Git.

git clone

©

Clona um repositório remoto existente para a sua máquina local.

git status

📖

Mostra o estado atual do diretório e do staging area.

git add

✔️

Adiciona as mudanças para o staging area.

git commit

🤝🏻

Salva as mudanças do staging area como um novo snapshot no respositório local, possui uma mensagem descrevendo as mudanças.

git push

💨

Empurra as mudanças feitas no repositório local para o repositório remoto, geralmente hospedado no GitHub.

git pull

🧲

Puxa as últimas atualizações do repositório remoto para o repositório local.

git branch

🌿

Lista, cria, renomeia ou deleta branches do repositório local.

git checkout

🩺

Altera a branch atual.

git merge

🔀

Combina as mudanças feitas de uma branch em outra branch.

git diff

≠

Mostra a diferença entre duas branches, commits e arquivos.

git log

📜

Mostra o histórico de commits da branch atual.
